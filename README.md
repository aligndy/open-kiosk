
# 프로젝트 명: 2300만 시니어를 위한 레트로 키오스크 솔루션

### 1. 프로젝트 배경 및 문제 정의

*   **대한민국 인구 5000만 명 중 50대 이상 인구는 약 2300만 명으로 전체의 46퍼센트를 차지함.

*   **1975년 이전 출생자인 이들은 40대 중반 이후에야 스마트폰과 같은 터치스크린 및 버튼리스(Button-less) 환경에 본격적으로 노출되었음.

*   **2012년 스마트폰 보급률 50퍼센트 돌파, 2015년 88퍼센트로 세계 1위를 기록하는 급격한 디지털 전환 과정에서 시니어 세대는 현재의 키오스크 UX에 큰 불편함을 느끼고 있음.

### 2. 핵심 솔루션: 자판기 기반 레트로 UI
*   **시니어 세대에게 가장 친숙한 전자 기기는 1977년 도입되어 1988년 서울 올림픽을 기점으로 전국에 보급된 자판기(Vending Machine)임.

*   **자판기 이용에 어려움이 없는 시니어의 행동 패턴에 집중하여, 터치스크린 내에 아날로그 자판기 형태의 레트로 UI를 재현했음.

*   **버튼리스 환경에서 오는 심리적 장벽을 제거하고, 익숙한 조작 방식을 통해 디지털 리터러시 격차를 해소함.

## 주요 목표

### 2300만 시니어 세대를 위한 레트로 UI 기반 접근성 향상

*   **2300만 시니어 특화 레트로 UI: 1975년 이전 출생자 2300만 명에게 익숙한 1977년식 자판기 UI를 터치스크린에 구현하여 조작 실수를 방지하고 심리적 접근성을 높였음.

*   **AI 음성 가이드 및 맞춤형 추천: 음성 명령 시스템과 고객 데이터 분석 기반의 메뉴 추천으로 복잡한 선택 과정을 대폭 단축했음.

*   **메뉴 정보와 주문 기능의 분리: 상세 메뉴 정보 확인과 주문 결제 단계를 이원화하여 메뉴 정보 제공이 주문 뎁스를 길게 만드는 현상을 방지했음. 주문은 주문대로 간소화하고, 메뉴는 메뉴대로 홍보와 정보 전달을 자세히 할 수 있게 영역을 구분했음.

### 자영업자를 위한 AI 기반 관리 편의성 증진

*   **AI 메뉴 관리 및 이미지 생성 자동화: 사진 촬영 시 메뉴 정보를 자동 인식하고 AI가 홍보용 고품질 이미지를 직접 생성하여 등록 절차를 간소화했음.

*   **AI 다국어 지원: 실시간 번역 엔진을 통해 외국인 고객 대응을 위한 다국어 환경을 즉각적으로 구축하여 매장 운영의 범위를 확장했음.

이 프로젝트를 통해 자영업자의 운영 부담을 줄이고 2300만 명의 시니어가 디지털 격차 없이 편리하게 이용하는 상생 모델을 제시하고자 함.
